# Terminal based 3D renderer 

A software renderer/rasterizer made in C with no dependencies other than [termbox2](https://github.com/termbox/termbox2) for rendering text to the terminal and standard C libs.

https://github.com/user-attachments/assets/bac4bef0-328c-4381-b0e1-2922029a5975

## Building it locally
### (For linux)
### Pre-requisites
- gcc
- termbox2

Clone the repo

`git clone https://github.com/PGamer2951/tui-raster.git`

Change into the repo's directory

`cd tui-raster`

Compile it with make

`make build`

Run the renderer

`./tui-raster.o`
